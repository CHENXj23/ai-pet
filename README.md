# 🐱🐶 宠物星球日记 · Pet Planet Diary

欢迎来到属于你和毛孩子的秘密基地！在这里，每一张照片都是回忆，每一段文字都是爱 ❤️

## 🎯 我们能做什么？

✅ 记录主子今天又干了啥傻事  
✅ 给每张照片配上暖心小故事  
✅ 选一只虚拟伙伴陪你一起养宠  
✅ 时间轴自动整理成长轨迹  

## 🧩 技术揭秘

- 用 Vue 3 写界面，TypeScript 保安全
- 微信云开发搞定后台，不用买服务器！
- 图片自动压缩，省流量又省钱 💰
- 支持相册/拍照双模式上传
  


## 🚀 如何启动？

```bash
git clone https://github.com/yourname/pet-diary.git
cd pet-diary
npm install
npm run dev:mp-weixin
```
## 📈 性能优化
- 图片上传前强制压缩至 200KB 以内
- 列表分页加载，避免一次性拉取大量数据
- 云函数按需调用，减少无效请求
